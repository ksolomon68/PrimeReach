Last Updated: March 23, 2026  |  Standard: WCAG 2.1 Level AA & AAA  |  Platform: CaltransBizConnect Web Application

Our Commitment to Accessibility

The California Department of Transportation (Caltrans) is committed to ensuring digital accessibility for all users, including people with disabilities. CaltransBizConnect strives to provide an inclusive experience and continuously improves our platform to meet and exceed recognized accessibility standards.

We believe that every Californian deserves equal access to contracting opportunities, resources, and the tools needed to grow their business — regardless of ability.

Conformance Status

CaltransBizConnect conforms to the following standards:

✓ WCAG 2.1 Level AA ✓ WCAG 2.1 Level AAA (selected criteria) ✓ Section 508 ✓ ADA Title II

These standards are achieved through our built-in accessibility widget, semantic HTML structure, ARIA landmark regions, keyboard navigability, and compliance-tested color contrast ratios across all interface components.

Accessibility Features Provided

Our integrated accessibility widget (accessible via the ♿ button or Alt+A) provides the following adjustments:

Accessibility Profiles

  • Motor Impaired — Enhanced keyboard navigation, large cursor, stopped animations
  • Visually Impaired — High contrast mode, enlarged text, screen reader optimization
  • Color Blind — Desaturated display, highlighted links, forced underlines
  • Dyslexia — Dyslexia-friendly font, increased spacing, reading guide

Individual Adjustments

  • Screen Reader Optimization — ARIA labels and role improvements
  • High Contrast Mode — Black background with yellow text
  • Dark Mode — Intelligent color inversion
  • Desaturate — Reduces color intensity for color-blind users
  • Highlight Links — Yellow background on all hyperlinks
  • Dyslexia-Friendly Font — OpenDyslexic / Comic Sans fallback
  • Big Cursor — 2× cursor size for motor impairment
  • Reading Guide — Horizontal line that follows the mouse pointer
  • Reading Mask — Darkens the page except the active reading area
  • Stop Animations — Disables all CSS animations and transitions
  • Enhanced Keyboard Navigation — Visible focus indicators
  • Link Underline — Forces underlines on all hyperlinks
  • Scalable Text — 100% to 150% in 10% increments

Additional Accessibility Features (Sitewide)

  • Skip to main content link on every page
  • aria-current="page" on active navigation links
  • All form fields have associated <label> elements
  • All images have descriptive alt attributes
  • Decorative images are marked aria-hidden="true"
  • Colour contrast ratio ≥ 4.5:1 for all body text
  • prefers-reduced-motion respected globally
  • ARIA landmark regions on every page (main, nav, footer)

Technical Specifications

  • Browsers tested: Chrome 120+, Firefox 121+, Safari 17+, Edge 120+
  • Screen readers tested: NVDA 2023, JAWS 2024, VoiceOver (macOS & iOS)
  • Keyboard accessible: All interactive elements reachable by Tab/Shift+Tab
  • Widget shortcut: Alt+A to open/close · Escape to close
  • Settings persistence: User preferences saved in localStorage
  • Mobile: Fully responsive, tested on iOS Safari 17 and Android Chrome 120
  • No external dependencies — Accessibility widget is 100% vanilla JavaScript

Known Limitations

While we strive for full WCAG 2.1 AA conformance, some legacy third-party embedded content may not fully meet all criteria. We are actively working to resolve any known gaps. If you encounter a barrier, please contact us using the information below.

Feedback & Contact

We welcome feedback on the accessibility of CaltransBizConnect. If you experience any barriers or have suggestions for improvement, please reach out:

Email: SBEss@dot.ca.gov

Website: caltransbizconnect.org/contact

Response Time: We aim to respond within 2 business days.

If you are not satisfied with our response, you may contact the California Department of Rehabilitation or file a complaint with the U.S. Department of Justice ADA Information Line .
